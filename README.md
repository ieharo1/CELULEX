# 🟢 TST SOLUTIONS - CELULEX

**CELULEX** es una tienda online de celulares desarrollada por **TST Solutions** ("Te Solucionamos Todo").

---

## 📱 ¿Qué es CELULEX?

**CELULEX** es una aplicación web profesional para venta de celulares, con catálogo dinámico y panel de administración.

> *"Tecnología que funciona. Soluciones que escalan."*

---

## ✨ Características Principales

### 🛒 Tienda
- Página principal profesional y responsive
- Catálogo dinámico de productos
- Vista de productos atractiva

### ⚙️ Panel de Administración
- Login de administrador
- CRUD completo de productos
- Agregar, editar y eliminar productos
- Gestión de inventario

### 🎨 Diseño
- Diseño responsive
- Interfaz moderna e intuitiva
